OCI RAG Assistant

Enterprise-grade agentic Retrieval-Augmented Generation platform that scrapes Oracle Cloud Infrastructure documentation, builds vector indexes in PostgreSQL (pgvector), and delivers grounded, context-aware answers via LLM.

  • End-to-end RAG pipeline: scraping → embeddings → vector search → LLM
  • FastAPI backend with `/chat` and `/chat-stream` APIs
  • Session-based conversation memory and streaming responses
  • Fully containerized with Docker & docker-compose

🏗 Architecture Overview

From Streamlit UI to FastAPI and pgvector-backed RAG.

High-level flow
  • Streamlit UI for interactive chat experience.
  • FastAPI backend exposes `/chat` and `/chat-stream` endpoints.
  • Agentic RAG engine performs retrieval and context building.
  • Embedding model outputs 1536‑dimensional vectors.
  • PostgreSQL + pgvector stores and searches embeddings.
  • LLM generates grounded, context‑aware responses.
Tech stack
  • Python 3.11, FastAPI, Streamlit
  • PostgreSQL & pgvector for similarity search
  • OpenAI API & LangChain for LLM + orchestration
  • Docker & docker-compose for reproducible deployment

🚀 Getting Started Locally

Minimal steps extracted from the project README.

Full setup guide ↗
  1. Create your .env file with at least:
    OPENAI_API_KEY=your_api_key_here
  2. Start the Dockerized stack:
    docker-compose up --build
  3. Ingest Oracle documentation into the vector store:
    docker exec -it oracle-rag-assistant-v12-app-1 python -m ingestion.run_ingestion
  4. Access the application:
    Streamlit UI: http://localhost:8501
    FastAPI docs: http://localhost:8000/docs
